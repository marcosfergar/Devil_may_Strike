# Devil May Strike

## Project Description
Devil May Strike is a thrilling action-adventure game where players embark on a journey filled with challenges and exciting gameplay. The game features stunning graphics, engaging storylines, and a variety of characters.

## Features
- **Engaging Gameplay**: Experience immersive gameplay that keeps players on the edge of their seats.
- **Multiple Characters**: Play as different characters, each with unique abilities and storylines.
- **Stunning Graphics**: Enjoy visually appealing graphics that enhance the gaming experience.
- **Challenging Levels**: Navigate through various levels that provide unique challenges and enemies.

## Installation
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/marcosfergar/Devil_may_Strike.git
   cd Devil_may_Strike
   ```
2. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the Game**:
   ```bash
   python main.py
   ```

## Usage
- Start the game using the command mentioned above.
- Follow the on-screen instructions to navigate through menus and gameplay.
